PORTFOLIO SITE

På dette tema blev vi præsenteret og fik gennemgået forskellige filtyper, såsom .jpeg, .jpg, .svg, .webp og .gif. Vi fik gennemgået i undervisningen hvad forskellen var på de forskellige filer og hvorfor vi bruger dem.

Vi gennemgik også ophavsret i forhold til brug af billeder fra andre sider på nettet. Her blev vi præsenteret for både colourbox og unsplash, der har rettighedsfrie billeder.

Temaets indhold var skabt på forhånd og vores opgave var at sætte koden op i html og css, ud fra de udleverede wireframes og layoutdiagrammer. Vi havde løbende undervisning i html og css- herunder grids, Media queries og flexboks.
Vi brugte coolors.co/generate til at vælge et farvetema til sitet og fik introduktion til Google fonts, og hvordan vi installerer og inkorporere dem på sitet.

Vi skulle på baggrund af det udleverede materiale skabe et site om computere, der var responsivt, dvs. fungerede både på desktop og mobil. Vi lærte at arbejde med mobilfirst og bruge Media queries til at lave version til desktop. Media queries bruges til at sætte rundt om det indhold der skal fungere på et andet format.

Vi arbejdede med programmer som visual code - herunder forskellige extensions såsom prettier og validering af HTML og CSS, Filezila og adobe XD.
Visual code er et program der bruges til at indeholde kode og hvor der bla. kan oprettes css og html filer.

Moodboard

På dette tema havde vi både en temaopgave der bestod i et mobilsite der indeholdte udleveret tekst og billeder, sat op med html og css.

Styletile

Vi afsluttede med en studiestartsprøve der bestod i at sætte mobilsitet op ud fra udleverede wireframes og layoutdiagrammer. Wireframes bruges til at vide hvordan man vil sætte sit site op og layoutdiagrammer bruges til at vise sin HTML kodnings layout
